# Claude Code Alternatives

## Definition
Multi-LLM Claude Code alternatives that allow using various AI provider models within the Claude Code interface.

## OpenClaude

**Description**: Open-source alternative to Claude Code that supports all LLM AI provider models.

**Discovered**: Via YouTube video about Claude Code source code leak incident.

**Use Case**: When you want Claude Code experience with flexibility of different LLM providers.

## Comparison

| Feature | Claude Code | OpenClaude |
|---------|------------|------------|
| Model Support | Anthropic only | All LLM providers |
| Source | Proprietary | Open source |
| Cost | API based | API based |
