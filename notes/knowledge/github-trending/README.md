## Evaluation Framework

### Three Dimensions
- **A**: New Trending (growth indicator)
- **B**: Substantial Value (fundamentals)
- **C**: Medium-Large Scale (durability)

### Rating System
- ⭐️⭐️⭐️⭐️⭐️ (Top): A+B+C
- ⭐️⭐️⭐️⭐️ (High): A+B
- ⭐️⭐️⭐️ (Normal): B+C
- ⭐️⭐️ (Watch): Only A or only C
