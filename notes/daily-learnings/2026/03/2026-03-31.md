### gog CLI 認證流程整理

**情境**
- gog（Google Workspace CLI）已透過 OAuth 完成 Gmail 授權綁定
- token 存在 `~/.config/gogcli/keyring/`（file-based keyring）

**--remote flag**
- 當 server 沒有本機瀏覽器時使用（如 SSH 遠端操作）
- 缺點：需密碼解鎖 file keyring
- 需設定 `GOG_KEYRING_PASSWORD` 環境變數

**最乾淨的做法（可直接接螢幕的機器）**
1. 編輯 `~/.config/gogcli/config.json`，設定 `{"keyring_backend":"auto"}`
2. 本機瀏覽器完成 OAuth
3. 系統 keyring（GNOME/KDE）由桌面 session 管理，不需要密碼環境變數

### Music Project — Interval System + Harmony Analyzer

**Problem: LLM Cannot Self-Correct Without Auditory Feedback**
- LLM generates wrong pitch intervals — no way to hear and fix without human
- Solution：build Music21 harmony analysis pipeline

**New Tool: Harmony Analyzer (`analyze_harmony.py`)**
```bash
python3.14 analyze_harmony.py output/departures/song_full.mid
```
- Uses `interval.Interval.isConsonant()` to check every pair of simultaneous notes
- Distinguishes truly bad intervals (m2, tritone) from acceptable modern ones (P4, M7)
- Reports: which parts clash, at what offset, what the interval is

**Key Discovery: Drum Hi-Hat Problem**
- MIDI hi-hat = pitch 42 = F#2 — Music21 treats it as pitched note
- Created false "tritone F#2-B2" clashes in harmony analysis
- Fix: skip percussion parts in cross-part interval checking

**Key Discovery: Part Ordering Bug**
- All parts inserted directly into Score with `.insert()` — no Part hierarchy
- MIDI output collapsed to 1 track → could not distinguish instruments
- Fix: build one Part per instrument, insert Parts into Score

**Interval-Based Chord System (Working)**
- Chords specified as names: 'D', 'Dmaj7', 'Bm'
- `build_chord(name, root_octave=N)` computes correct pitches from semitone intervals
- `roman_to_chord_name()` converts I/IV/V to actual chord names for any key
- Key-agnostic: change KEY setting → all chords recompute correctly

**Register Separation (departures.py v4)**
- Bass: D2-A2 (MIDI 38-45)
- Piano: D4+ (MIDI 50+)
- Strings: D5+ (MIDI 62+)
- Melody: D5+ (MIDI 62+)
- Drums: MIDI 36/38/42/46/49 (percussion, no pitch)
- Goal: no overlapping voices → reduced clashing
