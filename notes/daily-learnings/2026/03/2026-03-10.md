## Notes

### Instagram Graph API (DIY)
- Instagram Graph API doesn't accept direct binary file uploads from computer
- Requires a **publicly accessible image/video URL**
- Workflow required:
  1. Generate image
  2. Upload to S3/Cloud storage (e.g., Cloudflare R2)
  3. Get public URL
  4. Send to IG API
  5. Confirm IG fetched the image
  6. Delete cloud image (cleanup)

### Cloudflare R2 as Image Hosting
- Use Cloudflare R2 as image bed
- Upload via Wrangler CLI
- Reference: [Cloudflare R2 Setup Guide](https://vocus.cc/article/68f9ef0bfd89780001fe8eae)
