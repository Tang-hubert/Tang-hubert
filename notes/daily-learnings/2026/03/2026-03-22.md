### OpenClaw AI Agent

#### 記憶體機制
- OpenClaw 中，所有重要資訊若未動到 system prompt，`.openclaw/workplace/*` 下的內容只是 LLM 的回覆，並非真正記住
- 這意味著需要主動更新 system prompt 才能讓 AI Agent 真正記憶重要資訊

### i-En Instagram AI 專案

#### 專案結構
```
project_ien/
├── .gitignore
├── package.json
├── package-lock.json
├── .env                 # (需手動建立) 金鑰與安全變數
├── monitor.js
├── data/
│    └── memory.json     # (程式自動生成) FIFO 滑動視窗短期記憶庫
├── logs/
│    └── ien_system.log  # (程式自動生成) 系統運行日誌
└── src/
    ├── config.js       # 環境變數載入器
    ├── storage.js      # Sliding Window FIFO 管理模組
    ├── brain.js        # 大腦 (LLM) 模組
    ├── vision.js       # 視覺 (FLUX) 模組
    ├── publisher.js    # 發布 (IG API) 模組
    └── main.js         # 主控台排程入口
```
