### OpenClaw Not Using Tools Fix
- Symptom: OpenClaw only chats, won't use tools after update
- Common cause: `tools.profile: messaging` or narrower tool policy
- **Fix**: Change `tools.profile.messaging` → `tools.profile.coding`
- Gives agent more permissions to do things
