## Notes

### OpenCode AI - agency-agents Setup

#### Problem: Color Hex Conflict
OpenCode uses hex color markers that conflicted with agency-agents color format.

#### Solution: Windows Workflow
1. Use **Git Bash** on Windows to run `.sh` files
2. Pull latest updates: `git pull` to sync week's bugs and new agents
3. Run `scripts/convert.sh` - filters agents for each AI platform format
4. Run `scripts/install.sh` - installs agents to platform default folders
5. For OpenCode (project-scoped), manually copy converted agents:
   - Source: `agency-agents/.opencode/agents/*`
   - Target: OpenCode's default agents directory

#### Key Insight
OpenCode is **project-scoped** - agents must be installed from project root with message:
`[!!] OpenCode: project-scoped. Run from your project root to install there.`
