## Notes

### OpenClaw Version Update (2026.4.2 → 2026.3.28 Rollback)

**Reason for rollback**: "duplicate plugin id" error in v2026.4.2

**Breaking Changes in 2026.4.2** (require `openclaw doctor --fix`):
- xAI search settings path: `core tools.web.x_search.*` → `plugins.entries.xai.config.xSearch.*`
- Firecrawl settings path: `core tools.web.fetch.firecrawl.*` → `plugins.entries.firecrawl.config.webFetch.*`

**New features in 2026.4.2**:
| Feature | Description |
|---------|-------------|
| Task Flow Persistence | Background orchestration survives restarts |
| Compaction notifyUser | Can disable "Compacting context..." message |
| before_agent_reply hook | Plugins can insert content before LLM reply |
| Copilot/GitHub routing fix | More stable routing for Copilot provider |
| ClawHub version compatibility | Skills install no longer fails for v2026.3.22+ |

### Work-in-Midnight Dispatcher Architecture Fix

**Problem**: Trello move sometimes skipped due to Worker timeout/error

**Solution**: Moved Trello move from Worker (unreliable) to Dispatcher Phase 3.5 (deterministic)

**Worker changes**:
- Step 4: Remove Trello move
- Step 5: Add `trello_move_status: N/A (handled by dispatcher)`
- Security boundary: Discord tool and Trello API write forbidden

**Dispatcher Phase 3.5**:
```bash
for card_id in {processed_card_ids}; do
  for i in 1 2 3; do
    result=$(curl -s -X PUT "https://api.trello.com/1/cards/$card_id?key=$TRELLO_API_KEY&token=$TRELLO_TOKEN" \
      -d idList=69cd2aa475b5f0d4b1be3403)
    # Retry logic: max 3 attempts, 2s interval
  done
done
```

**Phase 6 Discord notification**: Includes Trello move status block (success N / failed M)

### Daily Summary Cron Job Update

**New Step 4 - Isolated Session File Merge**:
- Scans `memory/` for `{YYYY-MM-DD}-{slug}.md` isolated session files
- Merges content into daily-summary as `#### {slug}` subheading
- Merged content auto-pushed to GitHub

**Git Upload Scope**:
- `memory/YYYY-MM-DD.md` (within 3 days)
- `daily-summary/YYYY-MM-DD.md` (includes merged isolated session content)
- `backup/` - unrelated to Git

### Mis-sent Prompt Handling
- Discord edit **cannot** change already-sent prompt (OpenClaw sends to API on receipt)
- Correct approach: immediately follow up with "抱歉打錯，正確內容是：..."
