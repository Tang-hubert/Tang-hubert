### Work-in-Midnight 自動化研究系統
- **Purpose**: Automated research on Trello Pending cards at 4:00 AM Taiwan time
- **Architecture**:
  - Python script: `work_in_midnight_research.py`
  - Trello Board: `openclaw-cowork` (Pending List ID: `69cd2aa475b5f0d4b1be3401`)
  - Cron Job: isolated session, 30-minute timeout (increased from 10 min)
  - Discord delivery to `#openclaw-work-in-midnight`
- **Card Categories**: GitHub Repo / System Package Audit / Topic Research / Technical Implementation
- **Relay Mechanism** (key feature):
  - Each research session appends to existing `.md` files (not overwrite)
  - "待續..." (To be continued) markers for seamless session continuity
  - `research_progress.md` tracks all progress
  - Future sessions read progress → find card `.md` → continue from marker
- **Trello Card Flow**: Pending → on-work (start) → complete (finish)
- **Key Insight**: Phase completion triggers immediate disk write (append mode) to prevent context overflow
- **Timeout Lesson**: 10 minutes insufficient for research tasks; 30 minutes required
