### Work-in-Midnight × Notes Integration System

**Problem**: Two separate repos (work-in-midnight + Tang-hubert/notes) with no communication.

**Solution**: `wim_to_notes_sync.py` - Automated sync script

**Workflow**:
```
04:00 Work-in-Midnight research complete → research_progress.md updated
06:00 wim_to_notes_sync.py runs → sync to notes → push fork → create PR
Hubert merge PR → notes shows overnight research highlights ✅
```

**Script Logic**:
1. READ `research_progress.md` (read-only, never modify)
2. Parse `## YYYY-MM-DD` headers with timestamps
3. Find research entries where timestamp = yesterday (0-N entries)
4. Condense into table format
5. Write to `notes/daily-learnings/YYYY-MM-DD.md`
6. Push to fork → create PR to Tang-hubert/Tang-hubert
7. Announce PR link to `#work-in-midnight-details`

**Finding Format Support**:
- `- **topic**（第N次研究）：finding`
- `1. **topic**（第N次研究）：finding`
- `- **topic**（接續）：finding`
- `- **topic**：finding`

**Key Principle**: `research_progress.md` is READ ONLY - never modified by sync script
