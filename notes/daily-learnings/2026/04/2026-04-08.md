### Discord Channel Name vs ID Routing

**Key Insight**: Discord routing uses **Channel ID**, not channel name.

| Cron Job | Delivery | Uses Name or ID? |
|----------|----------|-----------------|
| Hack Skill | `channel: discord` | ❌ String `discord` |
| i-En Monitor | `channel: 1485058398486532167` | ✅ Channel ID |
| 每日總結 | `channel: discord` | ❌ String `discord` |
| Moltbook | `channel: discord` | ❌ String `discord` |

**Finding**: All cron jobs use ID or account-level routing, not channel name. Renaming `#for-free` → `#free-used` is safe and does not affect any cron job operations.

**Note**: OpenClaw cannot rename Discord channels directly - requires `Manage Channel` permission and Discord API access.
