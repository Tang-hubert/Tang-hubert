### Discord Inbound Worker Timeout Issue

**Problem**: When running long PinchBench tasks, Discord inbound worker times out after 30 minutes: `Discord inbound worker timed out`

**Resolution Process**:
1. Asked to放宽 → changed `channels.discord.inboundWorker.runTimeoutMs` to 7200000 (2 hours)
2. Hubert said other projects running, shouldn't change global value → reverted to 1800000
3. Tried using isolated cron job (sessionTarget: "isolated")
4. Multiple isolated sessions terminated due to parent session interruption (SIGTERM/SIGKILL)
5. Final solution: **nohup method** to decouple benchmark process from session

**Final Solution**:
```bash
nohup uv run python scripts/benchmark.py ... > bench_run.log 2>&1 &
```
