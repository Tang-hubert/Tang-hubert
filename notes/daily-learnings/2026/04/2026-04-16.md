### Work-in-Midnight: card-index.csv + Slug Naming Rules
- Slug naming rules (approved 04-15):

  - Chinese: **keep original** (e.g., `痛苦慾望`)

- Fields: card_id / card_name / slug / status / research_type / labels / target_dir / notes

### PinchBench vs Main Agent Testing
- PinchBench spawns isolated subagent = tests "brtclaw in lab conditions"
- Real daily session = brtclaw with context, memory, USER.md, live feedback
- These are fundamentally different evaluation approaches
- Main agent testing options:
  1. Custom Python script with sessions_send (handles multi-step tasks)
  2. "Cheating method": copy main session context as subagent system prompt
  3. Manual spot-check (most practical)
