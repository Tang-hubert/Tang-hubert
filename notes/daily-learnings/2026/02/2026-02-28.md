## Notes

### What is AstroNvim?
AstroNvim is a neovim configuration framework that provides pre-configured setups with plugin management.

### Installation Steps
1. Installed prerequisites (neovim, git, node.js)
2. Cloned AstroNvim configuration
3. Configured plugin manager (lazy.nvim)
4. Set up personal configurations

### Configuration Highlights
- Added plugins for Python development
- Added plugins for JavaScript/TypeScript
- Configured key mappings:
  - `<leader>pv` - File explorer
  - `<leader>fg` - Live grep
  - `<leader>ff` - Find files
