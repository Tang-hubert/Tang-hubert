### Key Meta IG API Knowledge

| Knowledge Point | Description |
|-----------------|-------------|
| Instagram Token Format | `IGAAU...` → use with `graph.instagram.com` |
| Facebook Token Format | `EAA...` → use with `graph.facebook.com` |
| Roles Token | Bypass OAuth, generated directly from Meta Developer, valid 60 days |
| OAuth Token | Requires App published + Instagram platform integration, bypass needs Business verification |
| `Invalid platform app` | Using Facebook App ID/Secret to call Instagram API, or App missing Instagram integration |
| Dev Mode Limitation | `instagram_content_publish` requires Business verification for full use (but Roles Token already has this permission) |
