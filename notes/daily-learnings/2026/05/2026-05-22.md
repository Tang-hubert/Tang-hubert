## Notes

### Threads Format Upgrade

**Background:** Hubert asked to analyze if Threads posts were in a format the public likes, and to fix them.

**Key Research Findings (Threads Viral传播 Key):**
- Images get **60% more engagement** than text-only links
- Threads users scroll fast — first 2 lines determine success
- Algorithm rewards replies/interactions, requires CTA (Call to Action)
- Hook must be dynamic, story-specific — generic hooks don't work

### Problems Fixed

| Problem | Old Format | New Format |
|---------|------------|------------|
| Weak Hook | Every post used "讓工程師停下來的理由：點進看 HN 社群怎麼說" | Dynamically generate story-specific hook |
| Caption Too Long | 150-200 chars with generic copy | Controlled within 120 chars |
| No CTA | Passive statements only | Unified "你怎麼看？" / "你在用嗎？" |
| GitHub `who` Too Long | Full report content written | Truncate at first period |
| GitHub `grade` Too Long | Full analysis report | Only star rating like `⭐⭐⭐/5` |
