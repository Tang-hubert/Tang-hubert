### OpenClaw Release Tracker Fix (Cron Job ID: 85a00b06)

**Problem:**
- `check_release.py` only posts headers (version, date, URL), completely missing Features/Bug Fixes content
- Past 60 versions all posted only headers, causing Hubert to receive no substantive highlights
- Cron job payload only had `python3 check_release.py`, no translation or Discord posting logic

**Fix:**
1. **Emergency manual re-post** of 2026.5.27 and 2026.5.26 stable versions with complete content (including Features/Bug Fixes)
2. **Updated Cron Job payload** to use isolated agentTurn, including complete Chinese translation flow:
   - Execute `check_release.py` to get new version
   - Read `.md` file, translate to Traditional Chinese
   - Post to Discord channel `1494120970703863948` (#openclaw-updates)
   - Format: `✨ 新功能` / `🐛 錯誤修正` / `📌 其他更新`
   - Each message no more than 1900 characters
   - Silent no-post when no new version
