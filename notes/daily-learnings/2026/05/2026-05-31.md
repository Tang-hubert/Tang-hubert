## Notes

### External-User LINE Bot Security Architecture (Cloudflare Worker)

**Completion Time:** 21:55 GMT+8

**Root Cause:**
Cloudflare Worker is deployed on Cloudflare servers, its `127.0.0.1` is its own localhost and cannot connect to Hubert's OpenClaw Gateway (`127.0.0.1:18789`).

**Solution:**
Change `OPENCLAW_GATEWAY_URL` from `http://127.0.0.1:18789/` to `https://openclaw.brt1122.com`

**Final Architecture:**
```
LINE → Cloudflare Worker (驗證+限流) → https://openclaw.brt1122.com → OpenClaw Gateway
```
